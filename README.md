# S3-Upload-Inotify

Script for uploading files to AWS S3 when write operation done.

## To start script
```
chmod +x uploads3
./uploads3
```
### Please configure awscli
```
aws configure 
```
```
<AWS ID>
<AWS KEY>
<AWS REGION>
<FILE FORMAT>
```

#### Please configure paths in the script

### Install aws-cli
```
sudo apt-get install -y python-pip
```
```
sudo pip install awscli
```
